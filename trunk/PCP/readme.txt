To install:
1) Mod_rewrite must be enabled in Apache
2) Create an empty MySQL database
3) Edit \config\database.php and enter your database information.
4) Edit .htaccess to make sure RewriteBase is pointing to the correct folder
5) Edit bootstrap.php and make sure $app_path is pointing to the correct folder
6) Navigate to the url \admin\install 
7) Enter a Admin username and password then click install
